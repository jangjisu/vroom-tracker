# 제품 방향

이 문서는 `vroom-tracker`가 현재 무엇을 만들고 있으며, 기능 판단에서 무엇을 우선하는지 기록한다.
사용자 유형과 아직 검증되지 않은 행동 가설은 `USER_INSIGHTS.md`에서 관리한다.

## 현재 상태

- 고속도로 휴게소 위치와 기본 정보를 지도에서 탐색할 수 있다.
- 현재 기능은 휴게소 데이터를 수집하고 조회하는 기반을 만드는 단계다.

## 합의된 목표

- 현재 제품 범위는 고속도로 휴게소 정보 서비스다.
- 사용자는 이동 중 가까운 휴게소를 찾는 것보다, 이동 경로에 있는 휴게소를 비교해 들를 곳을 고른다.
- 초기에는 지도에서 휴게소를 탐색하고 상세 정보를 비교하는 기능을 완성한다.
- 먹거리, 입점 매장, 대표 메뉴, 휴게소 특색처럼 방문 결정을 돕는 정보를 우선한다.
- 편의시설, 주차 규모, 정비 여부와 같은 실용 정보도 함께 제공한다.
- 이후 출발지와 목적지를 기준으로 경로상 휴게소를 탐색하는 기능으로 확장할 수 있다.
- 미래 확장을 예상해 현재 필요하지 않은 계층이나 추상화를 미리 만들지 않는다.

## 데이터 확장 원칙

- 새로운 정보는 사용자가 외부 API 명세를 제공하거나, 사용자와 데이터 출처를 함께 조사한 뒤 추가한다.
- 확인되지 않은 정보는 추정하거나 임의로 생성하지 않는다.
- 관리자 등록, 사용자 후기 등 외부 API 이외의 데이터 수집 방식은 필요해질 때 별도로 결정한다.

## 미결정 사항

- 경로 탐색 기능에서 사용할 지도 및 경로 API
- 휴게소 비교와 추천 기준
- 사용자 저장, 후기, 방문 기록 기능의 도입 여부
